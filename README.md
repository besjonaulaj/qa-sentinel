# 🛡️ QA Sentinel - Issue Tracker

A real-time, dynamic Kanban board designed for QA teams to track bugs and feature requests. Built with **Vanilla JavaScript** and **Supabase**.

## 🚀 Features
* **Live Database:** All tickets are stored in a Supabase cloud database.
* **Dynamic Workflow:** Move tickets from 'To Do' to 'In Progress' and 'Resolved' instantly.
* **Urgency Indicators:** High-priority tickets feature a visual pulse animation.
* **Persistent Storage:** Data remains even after closing the browser.
* **Full CRUD:** Create, Read, Update, and Delete capabilities.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Backend-as-a-Service:** [Supabase](https://supabase.com/)
* **Deployment:** GitHub Pages

## 📂 How to Run
1. Clone this repository.
2. Open `index.html` in your browser (preferably using Live Server).
